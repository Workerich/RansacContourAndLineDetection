# RansacContourAndLineDetection
Обнаружение линий и контуров на изображениях с использованием **RANSAC** и сравнением с **методом наименьших квадратов (МНК)**. Проект демонстрирует устойчивость алгоритма к шумам и выбросам в сравнении с классической аппроксимацией.


## 📖 Введение
При обработке изображений часто требуется выделить геометрические примитивы - прямые линии, окружности, параболы - даже на сильно зашумлённых данных. Традиционный метод наименьших квадратов (МНК) даёт точный результат только при отсутствии выбросов. Если же на изображении присутствует много мусора (случайные пиксели, артефакты), МНК ошибается.

**RANSAC** (RANdom SAmple Consensus) - вероятностный итеративный алгоритм, устойчивый к выбросам. Перед его запуском изображение проходит предобработку - из него извлекаются потенциально значимые точки. Среди них возможно окажутся аномалии, но это не проблема: RANSAC специально разработан для работы с зашумлёнными данными и уверенно отделяет полезный сигнал от случайного шума.

В этом репозитории реализованы оба подхода для трёх типов моделей: **прямая, окружность и парабола**. Также реализован *экспериментальный* приём нахождения контуров с использованием RANSAC. Вы можете наглядно сравнить результаты и выбрать оптимальный метод под вашу задачу.


## 🔬 Алгоритм нахождения контуров

Сам по себе RANSAC не работает напрямую с пикселями - нужна предварительная отборка точек.
1. Изображение переводится в **HSV-формат**, из которого по каналу яркости (V) отбираются потенциально значимые пиксели (пороговая фильтрация).
2. **RANSAC** итеративно ищет наборы «хороших» точек, образующих линии - после каждой найденной линии уже использованные точки удаляются из выборки.
3. Опционально - фильтрация выбросов внутри каждой итерации для повышения точности аппроксимации.
4. Все найденные точки закрашиваются красным на результирующем изображении.


## 📐 МНК - метод наименьших квадратов

Классический детерминированный подход: минимизирует суммарное квадратичное отклонение всех точек от теоретической модели. Быстрый и точный на чистых данных, но чувствителен к выбросам - именно поэтому сравнение с RANSAC так показательно.


## 🚀 Возможности
- Обнаружение прямых, окружностей и парабол, произвольных контуров на изображении.
- Предобработка через V‑канал яркости формата HSV.
- RANSAC с адаптивным числом итераций (расчёт по формуле).
- Визуализация найденных линий и контуров.
- Сохранение результатов.


## 🖥️ Установка для Windows

### 1. Требования
- Python 3.8+
- Flask 3.0.0+
- Opencv-python 4.5.0+

### 2. Клонирование репозитория
```bash
git clone https://github.com/Workerich/RansacContourAndLineDetection.git
cd RansacContourAndLineDetection
pip install -r requirements.txt
python app.py
```
